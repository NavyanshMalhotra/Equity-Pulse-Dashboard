# 📈 Equity Pulse Dashboard
An interactive, dopamine-inducing dashboard for tracking your favorite equities.

## Features
- **🏠 Macro Overview**: Real-time pulse of global indices (S&P 500, FTSE, Sensex), VIX, and Bond Yields.
- **🚀 Dynamic Equity Tabs**: Each stock gets a dedicated deep-dive tab.
- **📉 CandleStick Charts**: Interactive Plotly charts for precise technical analysis.
- **🐳 Whale Radar**: Track institutional and whale movements for each equity.
- **🗞️ Breaking News**: Curated real-time news headlines with source links.
- **🧠 Copaw AI Insights**: Executive-level, factual insights powered by Gemini 1.5 Pro.

## Technology Stack
- **Frontend**: Streamlit
- **Data**: yfinance, DuckDuckGo Search
- **Visuals**: Plotly (Dark Mode)
- **AI**: Google Gemini API

## Setup & Deployment
1. **Repository**: Fork/Clone this repository.
2. **Hosting**: Link this repo to [Streamlit Community Cloud](https://streamlit.io/cloud) for a live dashboard.
3. **Secrets**: Add your `GOOGLE_CLOUD_API_KEY` to the Streamlit secrets management.
